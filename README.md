## MÔ TẢ
-  Đây là dự án FrontEnd với chủ đề là Todo App. <br>
-  Được xây dựng bằng các công nghệ như là Html, Css, Javascript, Firebase. <br>

## TÍNH NĂNG
-  CRUD: Thêm, Sửa, Xóa, Xem công việc
-  Dark/Light Mode: Chế độ sáng tối màn hình
